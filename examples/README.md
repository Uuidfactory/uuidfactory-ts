# Examples for uuidfactory-ts

This folder contains example scripts demonstrating how to use the **uuidfactory-ts** TypeScript client.

## Files

- `basic.ts` – Generates UUID v1, v4, v7, GUID, and validates a single UUID.
- `batch.ts` – Generates multiple UUIDs at once and validates them in batch.

## Usage

Run any example with:

```bash
ts-node examples/basic.ts
```

```bash
ts-node examples/batch.ts
```

Ensure you have TypeScript and ts-node installed:

```bash
npm install -g typescript ts-node
```
